# Tecnologias Assistivas

## Teclado Virtual (Onscreen Keyboard)
Para habilitar o teclado virtual na tela (Onboard), siga estes passos:

1.  **Acessar o menu:**
    * Vá pelo menu do painel superior (**Sistema** > **Preferências** > **Tecnologias Assistivas**).
    * **OU** utilize a barra de pesquisa do menu e digite `Assistive Technologies`.

2.  **Configurar:**
    * Marque a caixa **Enable assistive technologies** (Habilitar tecnologias assistivas).
    * Clique no botão **Preferred Applications** (Aplicativos Preferidos).
    * Selecione a opção **Onboard** como teclado de acessibilidade.
    * Marque a opção **Run at start** (Executar na inicialização) se desejar que ele inicie automaticamente com o sistema.
